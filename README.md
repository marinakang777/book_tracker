# Book Tracker

**Автор:** Ашрафова Дарья
Группа/курс: Python начальный уровень
Дата: Май 2026

## Описание

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Есть фильтрация по жанру и количеству страниц. Данные сохраняются и загружаются из файла books.json.

## Технологии

* Python 3
* Tkinter (GUI)
* JSON (хранение данных)

## Как использовать API GitHub

Приложение использует публичный GitHub  (https://github.com). 
Для запуска приложения склонируйте репозиторий: 

 https://github.com/marinakang777/book_tracker

запустите программу book_tracker

Примеры использования приложения book_tracker:

- Добавьте книгу через форму и нажмите «Добавить книгу».
- Используйте фильтры для поиска по жанру или количеству страниц.
- Сохраните данные кнопкой «Сохранить в JSON».
- Загрузите ранее сохранённые данные кнопкой «Загрузить из JSON».
